# 🌌 Acerca del proyecto

**Car Rental Management App** es una aplicación móvil desarrollada para automatizar la gestión de renta de vehículos, ofreciendo una interfaz moderna con animaciones fluidas, autenticación mediante Google y persistencia de estado utilizando Firebase.

La aplicación fue diseñada para:

- 🚗 Gestionar vehículos
- 👥 Administrar usuarios
- 🔐 Autenticación con Google
- 🌙 Soporte para Dark Mode
- 📱 Experiencia móvil moderna
- ⚡ Animaciones avanzadas
- ☁️ Sincronización con Firebase
- 🎨 Interfaces dinámicas

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/AppMovilRentaVehiculos.git
```

---

## 2️⃣ Acceder al proyecto

```bash
cd AppMovilRentaVehiculos
```

---

## 3️⃣ Instalar dependencias

```bash
flutter pub get
```

---

## 4️⃣ Configurar Firebase

Agregar archivos:

```bash
google-services.json
GoogleService-Info.plist
```

---

## 5️⃣ Ejecutar aplicación

```bash
flutter run
```
